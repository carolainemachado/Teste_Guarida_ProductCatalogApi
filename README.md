#### <h2>Descrição</h2>
  Este é um conversor de moeda desenvolvido com .NET Core e Dapper. A aplicação permite gerenciar um catálogo de produtos, oferecendo operações de CRUD (Create, Read, Update, Delete) através de uma API RESTful.


## Funcionalidades
- Gerenciamento de produtos (CRUD)
- Validação básica de dados (nome e preço)
- Documentação da API com Swagger

#### <h2>Tecnologias Utilizadas:</h2>
- .NET Core
- Dapper
- SQLite
- Swagger
  
#### <h2>Preparar o Ambiente:</h2>
- [Node.js](https://nodejs.org/) (v14 ou superior)

## Instruções
1. Clone este repositório.
2. Restaure as dependências (dotnet restore).
3. Inicie o servidor (dotnet run).
4. Acesse a documentação da API em (http:localhost:5000/swagger).
5. Para testar a API, você pode usar o Postman. Importe a coleção disponível.
